# ServiceLiveTrans

实时语音转录服务 - livetrans 的重构版本。

## 功能特性

- 支持外部 Whisper 或 FunASR API 进行语音识别
- 网页后台控制
- 聚会预约功能
- 转录稿存储
- 支持直播源输入
- 支持网页传输麦克风片段

## 技术栈

- Nuxt 4
- Vue 3
- TypeScript

## Setup

安装依赖：

```bash
pnpm install
```

## Development Server

启动开发服务器：

```bash
pnpm dev
```

## Production

构建生产版本：

```bash
pnpm build
```

本地预览：

```bash
pnpm preview
```
